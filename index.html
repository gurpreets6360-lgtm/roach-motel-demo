Dark Pattern Demo

Roach Motel

This dark pattern makes it easy to join a service but hard to leave.

Companies use visual design, button size, color, and hidden choices to push users toward the option that benefits the business.

Easy sign-up with one visible action
Hidden or less visible decline option
Long and frustrating cancellation process

The main action is bright and easy to notice. The decline choice is small and faded.

Step 1: Easy Entry

Premium Trial Offer

This page shows how the interface pushes the user toward the free trial. The bright green button is the easiest action to click.

Example: A subscription is quick to join, but hard to cancel later.

Psychological Tactic: Users feel encouraged to continue because the preferred option is more visible and attractive.

Impact: Some users subscribe quickly and may later struggle to leave before renewal.

Notice the unequal visual design between the two options.

Step 2: Difficult Exit

Cancel Subscription

Now the user faces a much more difficult process. This is where the Roach Motel pattern traps people.

Step 1: Confirmation The user is asked again if they are sure, creating delay.
Step 2: Reconsideration Another screen pressures the user to stay subscribed.
Step 3: Final Exit Cancellation only happens after multiple steps.

Solution: Cancellation should be as simple and visible as sign-up.
